Button
Buttons are also called as call to action. We have range of buttons and their states. You may use a tag or button element, you need to add respective classes, and you are good to go.
Contained Button
" Button components is available in four variants btn-primary, btn-secondary, btn-error, btn-success" 

           <button class="btn btn-primary">PRIMARY</button>
           <button class="btn btn-secondary">SECONDARY</button>
           <button class="btn btn-error">ERROR</button>
           <button class="btn btn-success">SUCCESS</button>
Icon Button
"Icons Buttons can be created, by using a class fontawesome icons. 'btn' class followed by 'btn-icon'. " 

                <button class="btn btn-icon">
                <i class="fas fa-cart-plus"></i>
                Add to cart</button>
                <button class="btn btn-icon-2">
                <i class="fas fa-trash"></i>
                Delete </button>
Toggle Button And Floating Action Button
"Toggle buttons used to change the theme. Floating-action buttons can be fixed to right-bottom corner. " 

                <button class="btn btn-toggle">
                <i class="fas fa-toggle-on"></i>
                </button>
                <button class="btn btn-floating">
                <i class="fas fa-plus-circle"></i>
                </button>
