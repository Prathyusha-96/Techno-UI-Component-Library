Grid

A grid layout provides a versatile layout to place elements on the page.There are two grid layouts available two and three column layouts.

Two column grid
Two column grid can be used by using the 'grid-two' class. You can copy html part from below code snippet.
Grid One
Grid Two
          
          <div class="grid grid-two">
          <div class="grid-item">Grid One</div>
          <div class="grid-item">Grid Two</div>
          </div>
Three column grid
Three column grid can be used by using the 'grid-three' class. You can copy html part from below code snippet.
Grid One
Grid Two
Grid Three
              
              <div class="grid grid-three">
              <div class="grid-item">Grid One</div>
              <div class="grid-item">Grid Two</div>
              <div class="grid-item">Grid Three</div>
              </div>
Four column grid
