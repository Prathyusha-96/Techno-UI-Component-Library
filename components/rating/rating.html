Rating
Ratings provide insight regarding others' opinions and experiences, and can allow the user to submit a rating of their own.
Default Rating

Default Rating can be used by giving the class name default-rating to the div tag. Inside it, you can add the icons of your choice. Refer the HTML code snippet below. 

         
            <div class="default-rating">
                <i class="fas fa-star fa-2x" aria-hidden="true"></i>
                <i class="fas fa-star fa-2x" aria-hidden="true"></i>
                <i class="fas fa-star fa-2x" aria-hidden="true"></i>
                <i class="far fa-star fa-2x" aria-hidden="true"></i>
                <i class="far fa-star fa-2x" aria-hidden="true"></i>
              </div>
